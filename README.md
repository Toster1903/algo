# Algo Verifier

Простой локальный вебапп для проверки решений алгоритмических задач.

## Что умеет

- Выбор задачи из списка.
- Вставка Python-кода с функцией `solution(...)` или классом (для задач со структурами данных).
- Автопроверка на случайных тестах.
- Сравнение с эталонным решением из файла `zach.py`.
- Проверка времени выполнения (лимит на каждый тест).

Подключены все задачи из `zach.py`: `1-79`.

## Запуск

```bash
/usr/local/bin/python3 -m pip install -r requirements.txt
/usr/local/bin/python3 app.py
```

Открой в браузере:

- http://127.0.0.1:5000

## Запуск в Docker

```bash
docker compose up --build
```

Открой в браузере:

- http://127.0.0.1:5000

## Важно

- Код выполняется локально и не является полностью безопасной песочницей.
- Для задачи `29` используется отдельный встроенный эталон, потому что в `zach.py` там незавершенный `return`.
